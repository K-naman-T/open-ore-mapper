# Provenance

This repository is a clean open-source project for surface mineral signature mapping.

## Allowed Sources

- Public algorithms: Spectral Angle Mapper, non-negative least squares, raster tiling, numeric normalization.
- Code authored for the clean ore-mapper extraction work on 2026-06-01.
- Synthetic demo spectra generated from simple analytic curves.
- User-provided spectral libraries loaded at runtime.

## Excluded Sources

- Institution-specific branding, assets, logos, screenshots, or copy from private projects.
- Existing ML model weights, private datasets, HSI cubes, training results, or proprietary CSV spectra.
- Legacy monolith application code unrelated to the ore mapper core.
- Any real spectral library data without documented redistribution rights.

## First Release Policy

The first public release ships with synthetic demo spectra only. Real spectral libraries may be documented as user-provided inputs or optional download recipes after their licenses are reviewed.
